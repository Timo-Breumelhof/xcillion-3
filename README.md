# Xcillion-3 DNN theme - Bootstrap 3

> A community-maintained revival of the classic DNN Xcillion theme designed as the migration bridge to Xcillion-5, which will be based on Bootstrap 5.

**Xcillion-3** is a revival of the Xcillion theme that shipped with DNN 8 and was replaced by the Aperture theme in DNN 10. The version number makes the Bootstrap dependency explicit: **Bootstrap 3** — matching the original theme's foundation.

This project serves a dual purpose:

1. **Preserve** — Give DNN sites still running on Bootstrap 3 a maintained, starting point based on the familiar Xcillion foundation.
2. **Bridge** — Provide a clear migration path to **[Xcillion-5](https://github.com/Timo-Design/xcillion-5)**, the Bootstrap 5 counterpart. Because both themes share the same design language and naming convention, you can run them side by side on the same DNN installation and convert your pages one at a time — no big-bang migration required.

While the core look and feel stays true to the original, Xcillion-3 includes minor modernizations to remove outdated patterns and improve maintainability.

---

**Xcillion-5** — the full Bootstrap 5 version — is will be developed in a separate repository. 
Start there for new projects; use Xcillion-3 when you need a gradual, page-by-page upgrade path.
